# Paridala Yaswanth Kumar — Developer Portfolio

A modern, fully responsive developer portfolio website built with pure HTML, CSS, and JavaScript.

---

## 📁 Project Structure

```
portfolio/
│
├── index.html          ← Main HTML file (open this in browser)
├── style.css           ← All styles, animations, and responsive layout
├── script.js           ← Interactions: cursor, scroll reveal, counters, nav
│
├── images/
│   ├── civic-seva.svg          ← Civic Seva project thumbnail
│   ├── homeplate.svg           ← HomePlate project thumbnail
│   ├── tidewise.svg            ← TideWise project thumbnail
│   ├── accident-alert.svg      ← Accident Alert project thumbnail
│   ├── rfid-attendance.svg     ← RFID Attendance project thumbnail

│
└── README.md           ← This file
```

---

## 🚀 How to Open

1. **Extract** the ZIP file to a folder on your computer.
2. **Open** `index.html` in any modern browser (Chrome, Firefox, Edge).
3. That's it — no build step, no npm, no server needed.

> **Tip:** For the best experience with `backdrop-filter` glass effects, use Chrome or Edge.

---

## ✨ Features

- **Sticky glassmorphism navbar** with smooth scroll and active link highlight
- **Animated hero section** with floating gradient blobs and custom cursor
- **Scroll-reveal animations** on every section (fade-in + slide-up)
- **Animated stat counters** in the About section
- **Project cards** with image zoom on hover and glow shadow
- **Skill badges** with float hover effect
- **Certification cards** with animated top-border reveal
- **Education timeline** with gradient line and coloured dots
- **Contact section** with animated icon links + CTA card
- **Fully responsive** for desktop, tablet, and mobile
- **Mobile hamburger menu** with animated toggle

## 🛠️ Customisation

To update content, simply edit `index.html`:
- Change text inside the relevant section tags
- Update `href` attributes for social links
- Replace images in the `images/` folder (keep the same filenames)

To update styles, edit `style.css` — CSS variables at the top of the file control all colors and fonts.
