TideWise – Fishermen Safety Alert System

A multilingual web app providing real-time sea safety alerts and weather updates for fishermen. Displays the shortest safe route back to land during high tide or emergency situations.

HTML/CSS JavaScript Weather APIs Google Maps API

Accident Alert System – Vehicle Safety Device

A hardware-based safety system installed in vehicles that detects abnormal movement or accidents and automatically sends SMS alerts with GPS location to emergency contacts, police stations, and hospitals.

Arduino GPS Module GSM Module Embedded C

RFID Smart Attendance System

An RFID-based attendance system where tags on ID cards record student entry and exit in classrooms or exam halls. Displays real-time attendance records, roll numbers, and seat availability for administrators.

Arduino RFID Module Embedded C Serial Communication
